## ✨ Core Features

### 여행 계획 작성
- 시각적 UI를 통한 직관적인 일정 작성
- 드래그 앤 드롭으로 일정 순서 변경
- 구글맵 지도를 통한 날짜별 동선 확인

### 여행 계획 공유
- 완성된 계획 링크/카카오톡 공유
- 도시별 여행 계획 검색, 열람 및 북마크

### 사용자 경험
- 반응형 디자인
- 모바일 브라우저 및 터치 동작을 고려한 UX/UI
